UNIX

In the mid-1960's MIT, Bell Labs (AT&T), and General Electric (surprisingly made computer hardware) were working
on creating a system called Multics. Multics was a time-sharing operating system for the
GE-645 mainframe computer. There was frustration about the progress of this project
so many Bell Labs personal left the project. Bell labs was still interested in this type of
system so the lab worked to create a different system (Check out this old archive from AT&T). They named this system Unix which was a pun on Multics. This operating system
originally written in assembly language for the
PDP-11. In 1973, Unix was rewritten in C.
